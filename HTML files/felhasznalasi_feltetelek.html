Felhasználási feltételek

Üdvözöljük a [Receptoldal Neve] weboldalán! Az oldal használatával Ön elfogadja az alábbi feltételeket. Kérjük, figyelmesen olvassa el őket.

  1. Szolgáltatás Leírása

    A [Receptoldal Neve] egy online platform, amely minden étkezéshez receptötleteket kínál. Az oldalunkon található receptek API kulcs segítségével érhetők el, amelyek saját fejlesztésű és összeállítású receptek.

  2. Felhasználói Felelősség

    Az oldalt és annak tartalmát kizárólag saját felelősségére használja. Az oldalon található információk kizárólag tájékoztató jellegűek, és nem helyettesítik a szakmai tanácsadást.

  3. API Kulcs Használata

    Az API kulcs publikus, bárki számára hozzáférhető, és annak használata ingyenes. Az API kulcs segítségével lehívott receptek saját szerzemények, így azok további felhasználása, terjesztése vagy módosítása kizárólag a [Receptoldal Neve] előzetes írásos engedélyével lehetséges.

  4. Szerzői Jogok

    A weboldalon található összes tartalom, beleértve, de nem kizárólagosan a recepteket, képeket, szövegeket és grafikákat, a [Receptoldal Neve] tulajdonát képezik és szerzői jogi védelem alatt állnak. Ezeket a tartalmakat tilos másolni, terjeszteni, módosítani vagy bármilyen formában felhasználni a [Receptoldal Neve] kifejezett engedélye nélkül.

  5. Felelősség Korlátozása

    A [Receptoldal Neve] nem vállal felelősséget semmilyen közvetlen vagy közvetett kárért, amely az oldal használatából, az oldalon található információk felhasználásából, vagy az API kulcs használatából ered.

  6. Adatvédelem

    A felhasználók személyes adatainak védelme kiemelten fontos számunkra. Kérjük, olvassa el Adatvédelmi Szabályzatunkat, amely részletezi, hogyan kezeljük és védjük a felhasználók adatait.

  7. Módosítások

    A [Receptoldal Neve] fenntartja a jogot, hogy a Felhasználási Feltételeket bármikor módosítsa. A változásokat követően a weboldal használata az új feltételek elfogadását jelenti.

  8. Kapcsolat

    Ha bármilyen kérdése van a Felhasználási Feltételekkel kapcsolatban, kérjük, lépjen kapcsolatba velünk a [kapcsolatfelvételi e-mail cím] címen.

A [Receptoldal Neve] használatával Ön elfogadja a fenti feltételeket. Köszönjük, hogy minket választott!
